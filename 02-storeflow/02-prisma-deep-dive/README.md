# Module 02: Prisma Deep Dive

## Overview

Now that the project is scaffolded, it's time to master Prisma — your database toolkit for StoreFlow. This module takes you from basic CRUD to advanced queries, relations, and migrations.

| Concept | Why It Matters |
|---------|---------------|

| **Relations** | One-to-one, one-to-many, many-to-many with type safety |
| **Migrations** | Version-controlled database evolution |
| **Seeding** | Reproducible test/development data |
