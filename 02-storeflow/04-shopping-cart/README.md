# Module 04: Shopping Cart

## Overview

Cart architecture is one of the most underestimated challenges in e-commerce. Should guests have carts? What happens when a guest logs in with items already in their cart? This module covers session-based vs user-based carts, cart merging strategies, and atomic quantity updates.
