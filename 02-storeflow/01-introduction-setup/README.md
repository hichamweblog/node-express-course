# Module 01: Introduction & Setup

## Overview

Welcome to Course 2! You've mastered the fundamentals in DevJobs Pro — now it's time to tackle one of the most complex and rewarding domains in software engineering: **e-commerce**.

StoreFlow isn't a toy store template. It's a production-grade e-commerce backend that handles real money, real inventory, and real customer expectations. Every architectural decision in this course comes from battle-tested patterns used at companies like Shopify, Stripe, and Amazon.

## What You'll Learn

This module establishes the foundation:

- **E-commerce architecture** — Why online stores are architecturally different from other web apps
- **Prisma ORM** — Your new database toolkit (and why we're learning a second ORM)
- **Docker Compose** — Reproducible development environments with PostgreSQL + Redis
- **Project scaffold** — Production-ready Express 5 + TypeScript + Prisma setup

## Prerequisites

- ✅ Completed Course 1 (DevJobs Pro) or equivalent experience
- ✅ Comfortable with Express 5, TypeScript, and REST APIs

## Key Concepts Introduced

| Concept | Why It Matters |
|---------|---------------|
| **Domain-Driven Design** | E-commerce has well-defined domains (catalog, cart, orders, payments) |
| **Prisma vs Drizzle** | Different ORMs solve different problems — learn both, choose wisely |
| **Docker Compose** | Real teams use containerized environments. No more "works on my machine" |
| **Service Layer Pattern** | Business logic belongs in services, not controllers |
