# Module 08: MongoDB + Mongoose (Theory Module)

## 🎯 Module Overview

This module teaches MongoDB and Mongoose concepts through standalone examples. **DevJobs Pro uses PostgreSQL + Drizzle** (Module 09), but understanding MongoDB helps you choose the right database for future projects.

## 📚 Lessons

| #   | Lesson                                                             | Topics                                                      | Duration |
| --- | ------------------------------------------------------------------ | ----------------------------------------------------------- | -------- |
| 1   | [NoSQL Concepts & When to Use](./01-nosql-concepts-when-to-use.md) | NoSQL vs SQL, Document DBs, CAP Theorem, Decision Framework | 45 min   |
| 2   | [Mongoose Schemas & Models](./02-mongoose-schemas-models.md)       | ODM Concepts, Schema Definition, Validation, TypeScript     | 60 min   |
| 3   | [CRUD Operations with Mongoose](./03-crud-operations-mongoose.md)  | Create, Read, Update, Delete, Query Patterns, Pagination    | 60 min   |
| 4   | [Relationships & Population](./04-relationships-population.md)     | Embedding vs References, populate(), Performance            | 50 min   |

## 🔑 Key Takeaways

After completing this module, you'll understand:

- When to choose MongoDB vs PostgreSQL for your projects
- How to model data with schemas and validation
- CRUD operations with Mongoose
- Relationship patterns in document databases
